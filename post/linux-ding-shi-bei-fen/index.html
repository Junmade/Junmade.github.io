Linux定时备份

2019-05-29

  1. 创建目录

    su - oracle
sqlplus / as sysdba

## 创建逻辑目录
create directory bakTemp as '/data/bak';
## 查看所有目录
select * from dba_directories;

quit;

  2. 备份数据

    expdp username/password schemas=schemas dumpfile=expdp.dmp DIRECTORY=bakTemp;

  3. 恢复数据

    impdp username/password schemas=schemas dumpfile=expdp.dmp DIRECTORY=bakTemp;

  4. 定时

    ## 查看定时任务
crontab -l
## 删除定时任务
crontab -r
## 创建定时任务
crontab file.sh
crontab -e 命令

  5. 定时备份

    crontabCommand.sh

## 获取当前时间 YYYYmmdd
current_date=`date +%Y%m%d`
## 加载配置文件
## source /etc/profile
## 执行备份命令
expdp username/password schemas=schemas dumpfile=expdp.dmp DIRECTORY=bakTemp

    crontabRule.sh

## 每天晚上 2点执行任务
0 2 * * * /data/bak/crontabCommand.sh

    ## 执行 crontab
crontab ./crontabRule.sh
